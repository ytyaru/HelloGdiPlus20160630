# このソフトウェアについて #

HelloGdiPlus20160630は私個人が学習目的で作成したソフトウェアです。

WindowsAPIを使って簡単なGraphicsを表示しました。
GDI+のGraphicsオブジェクトを使っています。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
